No todos los archivos que agregas a un proyecto deberían ir a un repositorio, por ejemplo cuando 
tienes un archivo donde están tus contraseñas que comúnmente tienen la extensión .env o cuando te 
estás conectando a una base de datos; son archivos que nadie debe ver.

-Esto sirve para no subir archivos al repositorios(mas que todo imagenes o archivos bonarios
ya que no es buena practica)

COMO IGNORAR ARCHIVOS CON GIT IGNORE

-En el documento .gitignore se debe colocar la extension de los archivos a ignorar con un * o simplemente
con el nombre

EJEMPLO:

*.JPG 
anime.html
