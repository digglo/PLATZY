DEFINIR VARIABLES JAVASCRIPT:
let: tiene un alcance bloque, quiere decir que se puede accerder a esta variable desde cierta linea de
codigo hasta otra cierta linea de codigo

var: tiene un alcance global, quiere decir que se puede acceder a esta variable desde cualquier linea
de codigo

const: es una variable que ninca cambia y es global

DEFINIR UNA FUNCION JAVASCRIPT:
se utiliza FUNCTION

PROCESO DEBUGGIN JAVASCRIPT:
se utiliza console.log() y dentro de los parentesis  se coloca el valor que va a mostrar cuando 
ejecute la aplicacion

PROCESO DEBUGGIN PHP:
se puede utilizar ECHO, aunque la preferible es PRINT_R para imprimir lo que se traiga
